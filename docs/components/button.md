## Button-group
**预览**

<ClientOnly>
  <buttonGroup-demo></buttonGroup-demo>
</ClientOnly>

注意：**c-button-group的子元素应该是c-button**

**代码**
```
    <c-button-group>
      <c-button icon="left"></c-button>
      <c-button icon="right"></c-button>
    </c-button-group>
```
